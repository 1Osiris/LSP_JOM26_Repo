IntegerSet Class Documentation

A class that represents a set of integers. Each number can appear only once in the set.

How to Create a Set

Create an empty set:

IntegerSet set = new IntegerSet();

Create a set with numbers:

ArrayList<Integer> numbers = new ArrayList<>();
numbers.add(1);
numbers.add(2);
numbers.add(3);
IntegerSet set = new IntegerSet(numbers);

Basic Operations

Add a number:

set.add(5);

Remove a number:

set.remove(5);

Check if empty:

boolean empty = set.isEmpty();

Get size of set:

int size = set.length();

Clear all numbers:

set.clear();

Finding Values

Find largest number:

int max = set.largest();

Find smallest number:

int min = set.smallest();

Check if a number exists:

boolean hasNumber = set.contains(5);

Set Operations

Union (Combine two sets):

// Adds all numbers from setB to setA
setA.union(setB);

Intersection (Find common numbers):

// Keeps only numbers that are in both sets
setA.intersect(setB);

Difference (Remove numbers):

// Removes numbers that are in setB from setA
setA.diff(setB);

Print the Set

Convert to string:

IntegerSet set = new IntegerSet();
set.add(1);
set.add(2);
System.out.println(set); // Prints: [1, 2]
